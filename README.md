# IPL-Data-Visualization
IPL Data Visualization project using Python, Pandas, and Matplotlib to analyze IPL player statistics with interactive charts, performance insights, and data analysis.

## 📌 Project Overview
This project analyzes IPL player statistics using Python, Pandas, and Matplotlib. It performs data preprocessing, exploratory data analysis (EDA), and visualization to identify player performance trends and generate meaningful insights.

## 📂 Dataset
- IPL Player Statistics (2016–2025)
- Format: JSON

## 🚀 Features
- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Statistical Summary
- Correlation Analysis
- Data Visualization
- CSV Export
- Performance Insights

## 📊 Visualizations
- Bar Chart
- Line Chart
- Pie Chart
- Histogram
- Scatter Plot
- Box Plot
- Correlation Heatmap
- Virat Kohli Performance Analysis

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## 📈 Key Insights
- Identified players with the highest batting average.
- Compared strike rates across players.
- Analyzed bowling economy.
- Generated statistical summaries of IPL player performance.
- Visualized Virat Kohli's batting statistics.
